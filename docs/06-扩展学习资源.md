# 06｜扩展学习资源与下一步

## 1. 建议你继续掌握的知识图谱

## A. 控制理论基础

- 一阶/二阶系统响应（上升时间、超调、稳态误差）
- PID 与增益整定思路
- 离散控制与采样频率影响

## B. 飞行器与空气动力

- 固定翼的纵向/横侧向稳定性
- 侧滑角、偏航阻尼、方向稳定导数
- 舵面与阻力控制机理

## C. 传感器与状态估计

- IMU 噪声模型与标定
- 互补滤波/卡尔曼滤波基础
- 多传感器时间同步与坐标系处理

## D. 嵌入式实时系统

- 中断、定时器、总线仲裁
- 任务调度与循环实时性
- 数据记录与故障诊断

## 2. 推荐参考资料（公开可获取）

- PX4 / ArduPilot 官方文档（看架构、术语、调参流程）
- MIT OpenCourseWare 航空航天与控制课程（看理论框架）
- 《Small Unmanned Aircraft: Theory and Practice》（系统性很强）
- 各类固定翼调参实战文章（重点学习“实验方法”而非参数照抄）

> 建议：把“理论知识”与“当前项目代码”对照学习，学一点就回仓库验证一点。

## 3. 从 ArduFinless 到更完整飞控的路线

你可以按这个顺序升级：
1. 给本项目增加日志记录（串口 CSV 或 SD 卡）。
2. 建立离线分析流程（Python 绘图看 Beta/YawRate/输出关系）。
3. 增强控制律（例如分阶段增益、简单前馈）。
4. 再迁移到完整飞控框架（PX4/ArduPilot）理解工业化架构。

## 4. 结语

学飞控最重要的不是“背下多少术语”，而是形成工程闭环：
- 你能提出假设；
- 你能设计验证；
- 你能根据数据修正系统。

当你能稳定地做这三件事，你就已经走在飞控工程师的路上。
