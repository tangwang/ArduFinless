# 05｜安全、排障与飞行前检查

## 1. 安全底线（务必执行）

- 首次调试尽量无桨进行。
- 上桨测试必须固定机体，人员远离桨盘。
- 任何异常（抖动、舵面打满、传感器失联）立即断电检查。
- 保持急停/断电动作随时可执行。

## 2. 飞行前检查清单（可打印）

### 硬件
- [ ] 电池电压正常、供电稳定。
- [ ] 传感器固定牢靠，无松动。
- [ ] 舵机连杆无卡滞，舵面机械间隙可接受。
- [ ] 接线无反插、无虚接。

### 软件
- [ ] 参数已加载且符合当前机体。
- [ ] 串口 `params` 复核关键值。
- [ ] `report` 检查静止时传感器是否正常。
- [ ] RC 通道映射、方向、失效行为已验证。

### 现场
- [ ] 风况适中，场地开阔。
- [ ] 有观察员协助。
- [ ] 有明确的起飞—测试—降落计划。

## 3. 常见问题排障树

## Q1：上电后传感器初始化失败

检查顺序：
1. SPI 接线与 CS 引脚是否对应。
2. 电平转换、供电电压是否匹配。
3. 传感器是否损坏或接触不良。

## Q2：RC 输入无效或偶发失效

检查顺序：
1. 接收机输出是否在有效脉宽范围。
2. 输入引脚是否接到支持中断的通道。
3. 是否被 `checkTimeouts()` 判定超时（信号中断）。

## Q3：舵机输出方向不对

检查顺序：
1. 机械安装方向。
2. 输出通道是否接反。
3. 限位参数是否导致“看起来反向”。

## Q4：系统响应抖动

检查顺序：
1. 供电纹波与地线质量。
2. 机械间隙与舵机状态。
3. 滤波截止频率是否过高。
4. 增益是否过激。

## 4. “不能飞”的判定标准

以下任一满足都应暂停飞行：
- 传感器读数明显异常或跳变。
- 舵机无法稳定回中。
- 控制方向存在不确定性。
- 参数状态不可追溯（不知道当前到底设了什么）。

安全不是附加项，而是飞控工程的一部分。
